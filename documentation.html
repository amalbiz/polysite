Commands List

Here is a list of all the current commands available for PolyDiscordBot:

  • !commands - Lists out the commands available by the bot.
  • !ping - This will ping Poly.
  • !plays - This will play a song.
  • !poll: \"question\" \"option1\" \"option2\" \"option3\" \"option4\" - Creates a poll given question and options.
  • !recommend - Recommends a song from the top 50 songs of the day in the U.S. on Spotify.
  • !demographics - Lists the server roles and the number of members in them.

    • Below are Mod specific commands.

  • !history - Prints the command history of users.
  • !kick @member - This command will kick a member of the server.
  • !ban @member - This command will ban a member of the server.
  • !mute @member - This command will mute a member of the server.
  • !unmute @member - This command will unmute a member of the server.
